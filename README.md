# COVID-19 Clinical Case with Body Composition (BIA) Analysis

Exploratory analysis of body composition (BIA) data during the COVID-19 period using R. 

Analysis of a real-world dataset (n > 100) revealed a very strong negative correlation between body fat and body water (r = -0.99, p < 0.001), a strong negative correlation between BMI and body water (r = -0.96, p < 0.001), and a strong positive correlation between BMI and body fat (r = 0.96, p < 0.001). 

The data represent repeated measurements from a single individual over time; therefore, the observed relationships likely reflect underlying physiological dependencies and characteristics of BIA measurements. 
Tools and technologies: Noerden Minimi (bioelectrical impedance scale), Microsoft Excel, Google Sheets, Microsoft PowerPoint, RStudio
