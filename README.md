# BIA Body Composition Analysis During COVID-19

Exploratory analysis of body composition data (BMI, body fat, body water) from a home BIA device using R.

Results:

Fat vs Water: r = -0.99, p < 0.001

BMI vs Fat: r = 0.96, p < 0.001

BMI vs Water: r = -0.96, p < 0.001

Data represent repeated measurements from a single individual.
